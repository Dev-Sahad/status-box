# Security Policy

## Supported Versions

The table below outlines which versions of the status-box engine actively receive security updates and patches.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0.0 | :x:                |

## Reporting a Vulnerability

Security is highly valued in our open-source automation pipelines. If you discover a vulnerability, a configuration leak, or an input validation bypass within the parsing logic, please do not open a public issue.

### Process
1. Send a detailed report directly via email or private channels to the repository owner.
2. Include clear steps to reproduce the vulnerability or an example of a payload that bypasses `src/validator.js`.
3. A response and patch will be coordinated as quickly as possible.
